# Pomodoro Timer

A clean and responsive Pomodoro Timer built with HTML, CSS, and JavaScript.

The app includes customizable focus and break durations, automatic session switching, audio alerts, and a daily focus session history that persists after page reloads.


# Features

1. Start, pause, resume, and reset timer
2. Custom focus and break durations
3. Automatic transition between focus and break sessions
4. Audible notification when a session ends
5. Daily focus history stored in localStorage
6. Responsive design for desktop and mobile


# Tech Stack

 HTML
 CSS
 Vanilla JavaScript
